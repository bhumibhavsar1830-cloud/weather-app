## 📌 About

Weather App fetches real-time weather data for any city worldwide using the **OpenWeatherMap API**. Features a beautiful dark purple glassmorphism UI with current weather, humidity, wind speed, feels like temperature, and 5-day forecast.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🌤️ **Current Weather** | Temperature, description, weather icon |
| 💧 **Humidity** | Real-time humidity % |
| 💨 **Wind Speed** | Wind speed in m/s |
| 🌡️ **Feels Like** | Apparent temperature |
| 👁️ **Visibility** | Visibility in km |
| 📅 **5-Day Forecast** | Daily forecast with icons |
| 🏙️ **Quick Cities** | Mumbai, Delhi, Pune, Bangalore, London |

---

## 🛠️ Tech Stack

- **HTML5** — Structure
- **CSS3** — Glassmorphism dark UI, animations
- **JavaScript** — API calls, DOM manipulation
- **OpenWeatherMap API** — Real-time weather data (Free tier)

---

## 🚀 How to Run

### Step 1 — Get Free API Key
1. Go to [openweathermap.org](https://openweathermap.org/api)
2. Sign up for free
3. Copy your API key

### Step 2 — Add API Key
Open `index.html` and replace:
```javascript
const API_KEY = 'YOUR_API_KEY_HERE';
```
with your actual API key.

### Step 3 — Open in Browser
```
Just open index.html in any browser!
```
